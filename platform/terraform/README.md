# Terraform

## Purpose

This folder contains Terraform configuration for Civo infrastructure, including networks, firewalls, Kubernetes clusters, node pools, and backup storage.

## Does Not Belong Here

Application Deployments, Services, ConfigMaps, NetworkPolicies, Helm chart templates, and routine workload configuration do not belong here.

## Ownership

Terraform owns this area. Changes should be planned, reviewed, and applied with explicit state boundaries.
